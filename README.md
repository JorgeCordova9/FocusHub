# FocusHub
Focus hub, by students for students.
FocusHub: Your Intelligent Study Companion
FocusHub is a task-centric web application that transforms how students study. It combines time management, note-taking, visual collaboration, ambient music, and AI-powered assistance in one unified workspace.

Key Features:

Task Management: Organize study sessions with custom time allocations
Pomodoro Timer: Visual progress tracking with customizable intervals
Note Editor & PDF Import: Write and reference materials seamlessly
Digital Whiteboard: Sketch diagrams and solve problems visually
Smart Browser: Whitelist only essential websites to eliminate distractions
Focus Music: Curated playlists to set your study mood
AI Study Assistant: Real-time conversational AI powered by Groq's Llama 3.1
Progress Tracking: Visual goals and focus streak monitoring

Why It Matters:

Students waste time juggling multiple apps and fighting distractions. FocusHub solves this by creating a distraction-free workspace where everything needed for a study session exists in one place.
